Lesson 1: Basic Web Info & HTML


World Wide Web (WWW)


- HTML (Hypertext Markup Language) is main type of document on the World Wide Web
- All types of files can be found (i.e. plain text, html, video, images, music)

Major Pieces of the Web


- Computers run Web Browsers (i.e. Chrome, IE, Firefox, Safari) that uses HTTP (Hypertext Transport Protocol) protocol to communicate on the Internet to access resources/data on Web Servers

HTML (Hypertext Markup Language)


- HTML is made up of text (what you see), markup (what it looks like), and certain references if any (e.g. images, videos, or other docs) - HTML is comprised of structure with components called "tags"
- A complete HTML structure of a tag can be referenced as "element," with an opening tag and a corresponding closing tag
- Link to good video explanation here

Computers are Stupid


- Computers literally have to take instructions from what programmers give to them in code
- Small mistakes can lead to huge problems when given to "Stupid" computers

Inline vs Block


- Certain tags are considered inline tags (i.e. break tags, emphasis tags, and image tags) which end the line while block tags (i.e. paragraph tags) place a virtual box surrounding the content

Lesson 2: Creating Structured Documents with HTML


"Tree-Like Structure" of HTML


- HTML code can be seen as a tree-like structure due to fact that HTML elements can have other elements inside them forming "nested" boxes
- The more nested the HTML elements, the deeper the code can be viewed with deeper indentations

Tools Programmers Use to Write HTML Code


- Text editing programs (i.e. Sublime Text) make life easier for programmers
- Scratchpad.io text editor is great for immediate feedback and small projects
- Codepen.io text editor is great for sharing projects to the public or with others
- Sublime Text editor is more powerful with advanced features and versatile for larger projects

Lesson 3: Adding CSS Style & HTML Structure


What is CSS?


- Cascading Style Sheets (CSS) is code written to control "styles" of HTML elements
- "Cascading" means the most specific rule that describes the element that gets applied
- Inheritance is important concept and key feature in CSS by which properties are applied not only to specific elements, but also its descendants
- Several CSS files or sheets can be referenced in HTML

How to Include CSS Styling in a Web Page?


- Method 1: Write CSS in the (head) of HTML
- Method 2: Link your HTM to separate CSS file (most likely method)
- Method 3: Write CSS Style Inline with HTML

The Box Model and Positioning of Boxes


- CSS has structure in relation to boxes for efficient design and layout
- The Box Model consists of Margin, Border, Padding, and Content - The size of your HTML element is the sum of Border, Padding, and Content
- DIVs are Block elements, so by default they take up the entire width of a page
- Adding rule display: flex to appropriate CSS will override the above behavior and let DIVs appear next to each other
